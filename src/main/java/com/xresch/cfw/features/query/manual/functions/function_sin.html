Use this function to get the sinus value of a number.

Following query creates random records and uses the sin-function:

| source empty records=100
| set
	NUMBA=random(-88, 900)
	# round down to integer value
	SIN_NUMBA=sin(NUMBA) 
	# all following return 0
	NOTHING_RETURNS_ZERO=sin() 
	RETURNS_ZERO_AGAIN=sin(null)
	STRING_ZERO=sin('returns-0')
	BOOL_ZERO=sin(true)

Just for fun lets use some math-magic to create a sinus-wavey chart:

| source empty records=100
| set # do some math-magic to create a sinus-wave
	NAME="WAVE"
	INDEX=count()+1
	TIME= now(null, 0, 0, 0, INDEX)
	SINUS=sin(INDEX*0.3) 
	VALUE=(SINUS*100)*INDEX
| chart # display as a chart
	by=[NAME]
	type=bar
	x=TIME
	y=VALUE
