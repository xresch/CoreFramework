The move command is used to move fields before or after another field, for example to order columns in a table.

Following code will create random records, and moves the LIKES_TIRAMISU field before the ID field:

| source random 
| move before=ID LIKES_TIRAMISU

Following moves two fields after the ID field:

| source random 
| move after=ID TIME LAST_LOGIN

You can combine move with before and after, as well as define the fieldnames comma-separated or as an array:

| source random 
| move before=ID INDEX, LIKES_TIRAMISU
| move after=ID ["TIME", "LAST_LOGIN"]
