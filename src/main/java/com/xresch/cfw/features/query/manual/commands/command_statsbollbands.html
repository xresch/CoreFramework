Used to calculate Bollinger Bands values for a field. Can group the statistics by specific fields using the by-parameter.

The following statsbollbands example groups by WAREHOUSE AND ITEM and calculates the bollinger bands for the field COUNT: 

| meta 
	BY = ["WAREHOUSE", "ITEM"]
	FIELD = "COUNT"
| source random 
	type=series 
	seriesCount=1 
	records=300
| statsbollbands by=meta(BY) field=META(FIELD) period=20 precision=6
| chart
	by=meta(BY)
	x=TIME
	y=[META(FIELD), 'boll-upper', 'boll-movavg', 'boll-lower']
	ymin=null
	height="500"
