This command skips a certain amount of records at the end or beginning of all records.

Following query creates 10 records and skips the last:

| source random records = 10
| set INDEX = INDEX + 1
| skip

Following query creates 10 records and skips the last 3:

| source random records = 10
| set INDEX = INDEX + 1
| skip 3

Following query creates 10 records and skips the first 3:

| source random records = 10
| set INDEX = INDEX + 1
| skip 3 
	first = true
