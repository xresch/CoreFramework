This function allows you to encode strings to Base64.

Following code will create random series data and creates encoded Base64 values for links:

| source random records=100 type=series
| set 
	# encode a string
	ENCODED=encodeBase64(WAREHOUSE)
	# encode in URL concatenation
	LINK="https://cfw.fruittest.sample/query?id="+COUNT+"&warehouse="+encodeBase64(WAREHOUSE)+"&item="+encodeBase64(ITEM)
	# assign value of LINK to another field BUTTON
	BUTTON=LINK
| formatfield
	# create a button from the link
	BUTTON=['link', "Click!"]
