This command lets you change how the data is displayed.

The following query creates 42 random records and displays them as panels. 

source random records = 42
| display as=panels

Further you can set the fields that should be visible by providing an array of fieldnames. This might not work for all display options, for example the JSON display will always show all fields. 

| source random
| rename URL = url
| rename LAST_LOGIN = "Last Login"
| display as=biggertable  visible=[FIRSTNAME, LASTNAME, "Last Login", url]
