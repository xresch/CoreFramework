The applog source reads log events from this applications log files.

An example query that dedups the log data by url and shows the last 100 entries:

|source applog
| dedup url
| tail 100

Another example that filters the logs by server start messages:

| source applog
| filter message ~= "Started @"

If you want to only scan the logs in the last log file, add the parameter last:

| source applog scope=last
| filter level == "INFO"
