This command executes the given functions once per query. If you need to execute functions for every record, use the each-command.

Following query uses the command to offset the timeframe for a second query:

| globals multidisplay=2
| source random records = 10
| keep TIME
;
# offset earliest/latest by one day, alternatively you can use earliestSet() or latestSet()
| execute timeframeoffset(0,-1, "d")
| source random records = 10
| keep TIME
