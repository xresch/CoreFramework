The coalesce-function returns the first non-null value given as it's parameters.

The following example creates sample records and show some example uses of the coalesce-function:

| source random 
| keep LASTNAME
| set # add some more nulls for demonstration
	LASTNAME = if(random(0,100) > 50, LASTNAME, null)
| set
	LASTNAME = coalesce(LASTNAME, "N/A")

If you also need to handle empty strings, use the nullif function to convert the empty strings to null:

| source random 
| keep LASTNAME
| set # add some more nulls for demonstration
	LASTNAME = if(random(0,100) > 50, LASTNAME, '')
| set
	LASTNAME = coalesce(nullif(LASTNAME, ''), "N/A")
