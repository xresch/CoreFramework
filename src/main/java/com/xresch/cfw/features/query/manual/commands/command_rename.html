Renames the chosen fields. This command will change the order of the fields. Renamed fields will be put at the end of the list of fields.

Following code will rename three fields. 

| source random 
| rename URL=Link LASTNAME=Name

If a name contains blanks, put the name into single or double quotes. 

| source random 
| rename ID="Personal Identity"
| rename 'Personal Identity'="Personal ID"

If you give a field a name that already is taken by another field, the existing field is removed. Following will remove the field LIKES_TIRAMISU and LAST_LOGIN will be given the new name LIKES_TIRAMISU. 

| source random 
| rename LAST_LOGIN=LIKES_TIRAMISU
