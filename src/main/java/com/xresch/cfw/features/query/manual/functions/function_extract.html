Extracts a value using regular expression. The return value is defined using a regex group.

Good to know:

  • Multi Line Support:  The regular expressions can be used for strings containing multiple lines.
  • Dot Matches Newline:  The dot separator matches line break characters.

Following query creates example records and extracts various values:

| source random records=1000
| keep ID, URL
| set
	# extracts the middle part from the ID
	MIDDLE=extract(ID, ".*?-(.*?-.*?)-.*")
	# extracts the host from the url
	HOST=extract(URL, ".*?www.(.*?).url.*")
	# use group index if you have more then one group in your regular expression
	ID_FROM_URL=extract(URL, ".*?www.(.*?).url/.*?id=(.*)", 1)
