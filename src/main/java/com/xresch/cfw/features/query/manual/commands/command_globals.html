The globals command is used to add custom properties to all queries query. These global properties are case sensitive.

The following globals will fulfill specific functions:

  • multidisplay:  Number of columns used when displaying the results of multiple queries. Value must be between 1 and 6.

Multidisplay: Same Width Columns

Following example has 3 different queries separated by semicolon. The first query sets two globals and fetches some random data. The query shows also the difference between globals and metadata command. Each query sets the query specific metadata property 'name' to change the title of the query. 

| globals 
	multidisplay = 3 # define a number to set the number of columns
	myArray = ["A", "B", "C"] #used to set a custom parameter
	keepFields = [CUSTOM, VALUE] 
| metadata name="Aurora" | source random records=5 | set CUSTOM=randomFrom(globals(myArray)) | keep globals(keepFields)
;
| metadata name="Hera" | source random records=5 | set CUSTOM=randomFrom(globals(myArray)) | keep globals(keepFields)
;
| metadata name="Freia" | source random records=5  | set CUSTOM=randomFrom(globals(myArray)) | keep globals(keepFields)

Multidisplay: Custom Width Columns

To have custom widths for each column, define an array with numbers that define the ratio between the columns. This does not have to be specified with percentages, if you want a 3 to 1 ratio you can specify [3,1].

| globals
	multidisplay = [30, 70] # ratio for columns, here the first column is 30% and second is 70&
| source random records = 100 type="stats" seriesCount=3
# Make a Pie Chart
| chart 
	by = [STATISTIC]
	type = "pie" 
	aggregation = avg
	x = TIME
	y=[AVG]
	height = "500px"
	legendpos= bottom
	
; # Make a Line Chart
| resultcopy 
| chart 
	by = [STATISTIC]
	type = "area" 
	x = TIME
	y=[AVG]
	stacked=true
	tension=0.2

Multidisplay: Custom Rows with Custom Columns

Use "| globals multidisplay = true" and use "| meta multipercent = {percent}" to define a custom amount of columns every row. Each row get's filled up until it's width get's about 100%. Here is an example that makes rows with 1, 2, 3, and 4 charts:

| paramdefaults
	dataAndChart = `
		| source random type="stats" seriesCount=1 
		| chart  
			type="area" 
			x="TIME" 
			y=[MIN, AVG, P90, MAX] 
			height="300px" 
			tension = 0.4
		`
| globals multidisplay = true # set th

# 1 row 1 chart 100%
| meta multipercent = 100 $dataAndChart$ # insert the code above like this because we are lazy
# 1 row 2 charts 50%
; | meta multipercent = 50 $dataAndChart$
; | meta multipercent = 50 $dataAndChart$

# 1 row 3 charts 33%
; | meta multipercent = 33 $dataAndChart$
; | meta multipercent = 33 $dataAndChart$
; | meta multipercent = 33 $dataAndChart$

# 1 row 4 charts various widths
; | meta multipercent = 10 $dataAndChart$
; | meta multipercent = 20 $dataAndChart$
; | meta multipercent = 30 $dataAndChart$
; | meta multipercent = 40 $dataAndChart$
