The query language supports the creation and use of objects. This can be helpful in use cases like fetching data from sources returning JSON structures, or when using the API to execute queries and structuring the returned result.

Using Emojis in Queries

You can use emojis in queries. Following displays the status of records as emojis: 

| source random records = 100
| set

| formatfield emoji = ['css', "padding", "0px"]

	as=tiles 
	titlefields=[emoji] 
	visiblefields=[]

	# zoom & sizefactor used to hack away some padding
	zoom = 30
	settings = {sizefactor: 3}
