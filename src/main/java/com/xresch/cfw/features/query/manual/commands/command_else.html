The else-command is used to start an else- or elseif-block in an if-statement.

Simple example of an else-statement:

| source random records = 20
| if (VALUE < 50)
	| set SIZE = "Small"
| else
	| set SIZE = "Big"	
| end
| formatfield
	SIZE = ['case' 
				, "==Small", "cfw-green"
				, [], "cfw-red"
		   ]
| sort VALUE

You can add a condition to your else statement to turn it into an elseif-block:

| source random records = 20
| if (VALUE < 33)
	| set SIZE = "Small"
	
| else (VALUE < 66)
	| set SIZE = "Medium"
	
| else
	| set SIZE = "Big"
		
| end
| formatfield
	SIZE = ['case' 
			, "==Small", "cfw-green"
			, "==Medium", "cfw-orange"
			, [], "cfw-red"
			]
| sort VALUE
