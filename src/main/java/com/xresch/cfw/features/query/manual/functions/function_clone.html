Returns a clone for an array or json object. Useful if you want to avoid to modify the original value.

Following query uses the clone function and manipulates the ARRAY_NUMBERS to demonstrate the clone function:

| source random type=arrays records=5
| keep ARRAY_NUMBERS
| set
	#get first number
	FIRST_NUMBER=ARRAY_NUMBERS[0]
	#reference the array
	NOT_A_CLONE=ARRAY_NUMBERS
	#create a clone of the array
	NUMBERS_CLONE=clone(ARRAY_NUMBERS)
	#override element in array
	ARRAY_NUMBERS[0]=X

Following query uses creates a clone of the GRAPH_DATA JSON object and manipulates the original to demonstrate the clone function:

| source random type=various records=10
| keep GRAPH_DATA
| set
	GRAPH_DATA_CLONE=clone(GRAPH_DATA)
	GRAPH_DATA.rounded.x=round(GRAPH_DATA.x,2)
	GRAPH_DATA.rounded.y=round(GRAPH_DATA.y,2)
| filter 
	GRAPH_DATA.rounded.x <= 5
