The csv source takes csv data with a header line and converts it into records. By default it splits the text by the newline character ','.

An simple query that takes very simple csv input:

| source csv 
data=`
id,item
1,apple
2,orange
3, strawberry
`

You can define a separator and use quotes to include separators in values:

| source csv 
separator=";"
data=`
	id;item
	1;"tiramisu"
	2;"i;scream"
	3;"\"   panna cotta    \""
`

You can use multiple characters for a separator:

| source csv 
separator="---"
data=`
	id---character---boolean
	1---a---true
	2---b---false
	3---c---true
`
