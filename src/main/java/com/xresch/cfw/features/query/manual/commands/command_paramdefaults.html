The paramdefaults command is used to add default values for parameters. These parameter properties are case sensitive.

Following query adds default parameter values and uses them afterwards:

# RECOMMENDED: set default values for all parameters to always have a value.
# This also helps in case you want to copy and paste a query
| paramdefaults 
	text="default" 
	hello="hello world"
| source empty
| set
	# returns "default" if not set, parameter value otherwise
	TEXT_VALUE = param('text') 
	# returns "hello world" if not set, parameter value otherwise
	HELLO_VALUE = param('hello') 
	# returns null if not set, as there is no default, parameter value otherwise
	BOOLEAN_VALUE = param('boolean')
