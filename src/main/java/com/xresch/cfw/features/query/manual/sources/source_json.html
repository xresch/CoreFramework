This source is parsing a JSON string. It is useful if you need to parse custom data.

Following is a basic example how to use the source. use the parameter data to specify an array of :

source json data='
[ {"FIRSTNAME": "Serena", "LASTNAME": "Aloha",        "LIKES_TIRAMISU": true, "TIME": 1649957893200},
 ,{"FIRSTNAME": "Alena", "LASTNAME": "Konnichiwa",    "LIKES_TIRAMISU": false, "TIME": 1649957893200},
 ,{"FIRSTNAME": "Kassandra", "LASTNAME": "Ciaobella", "LIKES_TIRAMISU": true, "TIME": 1649957893200}
]'

To above command, we can add the parameter timefield to also be able to use the time range picker and filter by time. As the time is in epoch, this will work without specifying the time format. If you want to see the data, choose a time range that includes the 2022-02-02.

source json timefield="TIME" data='
[ {"FIRSTNAME": "Serena", "LASTNAME": "Aloha",        "LIKES_TIRAMISU": true, "TIME": 1643799764000},
 ,{"FIRSTNAME": "Alena", "LASTNAME": "Konnichiwa",    "LIKES_TIRAMISU": false, "TIME": 1643799764000},
 ,{"FIRSTNAME": "Kassandra", "LASTNAME": "Ciaobella", "LIKES_TIRAMISU": true, "TIME": 1643799764000}
]'

If the time is not in epoch milliseconds, we can specify the time format using the time format parameter:

source json timefield="time" timeformat="yyyy-MM-dd'T'HH:mm:ss.SSS" data='
