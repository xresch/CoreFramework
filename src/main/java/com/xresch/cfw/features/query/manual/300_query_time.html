This page gives you an overview how to work with time in queries.

Time Formatting

For formatting time, you can use the command formatfield. There are two formatters called 'date' and 'timestamp' to specify time formats, here a small example: 

| source random 
| set
	LOGIN_TIME = LAST_LOGIN
| formatfield 
	LAST_LOGIN=date
	LOGIN_TIME =[timestamp, "HH:mm:ss"]
	TIME=timestamp
| keep FIRSTNAME, LAST_LOGIN, LOGIN_TIME, TIME

Time Functions

The following functions are available to work with time:

  • now: Returns current time and offsets from current time.
  • earliest: Returns earliest time as epoch time or in a specific format.
  • earliestSet: Set the earliest time for the current query.
  • latest: Returns latest time as epoch time or in a specific format.
  • latestSet: Set the latest time for the current query.
  • timeframeOffset: Takes the earliest/latest times and offsets them by the specified amount..
  • timeOffset: Takes epoch milliseconds and offsets it by the specified amount..
  • timeParse: Parses time from a string representation and returns it as epoch milliseconds.
