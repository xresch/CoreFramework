The metadata command is used to add custom properties to a query.

The following properties will also be used by the query engine itself:

  • name:  Will be used to give the current query a name. Used to identify the query with other commands like 'resultcompare'.
  • title:  When set to true, the name will be shown as a title on the User Interface.
  • titleduration:  When set to true, the title will also contain the queries execution duration. (Default: true)
  • description:  Will be used as a description on the user interface.
  • template:  Set to true if query is a template to be used with mimic command. Query will be cancelled at the metadata command. Any results caused by the query will be removed.
  • multipercent:  Set to a percentage value to define the width of your result on a row. See below for an example.

Basic Example

Following code has two queries and adds the properties name and description to each query. The name property will also be used in the user interface for the title of the query. 

| metadata 
	name="Random Records" 
	title=true
	description="Query creates random records." 
	custom="whatever you wanna say."
| source random
;
| metadata 
	name="More Randomness" 
	title=true
	description="just another example query"
| source random

Multidisplay: Custom Rows with Custom Columns

Use "| globals multidisplay = true" and use "| meta multipercent = {percent}" to define a custom amount of columns every row. Each row get's filled up until it's width get's about 100%. Here is an example that makes rows with 1, 2, 3, and 4 charts:

| paramdefaults
	dataAndChart = `
		| source random type="stats" seriesCount=1 
		| chart  
			type="area" 
			x="TIME" 
			y=[MIN, AVG, P90, MAX] 
			height="300px" 
			tension = 0.4
		`
| globals multidisplay = true # set th

# 1 row 1 chart 100%
| meta multipercent = 100 $dataAndChart$ # insert the code above like this because we are lazy
# 1 row 2 charts 50%
; | meta multipercent = 50 $dataAndChart$
; | meta multipercent = 50 $dataAndChart$

# 1 row 3 charts 33%
; | meta multipercent = 33 $dataAndChart$
; | meta multipercent = 33 $dataAndChart$
; | meta multipercent = 33 $dataAndChart$

# 1 row 4 charts various widths
; | meta multipercent = 10 $dataAndChart$
; | meta multipercent = 20 $dataAndChart$
; | meta multipercent = 30 $dataAndChart$
; | meta multipercent = 40 $dataAndChart$
