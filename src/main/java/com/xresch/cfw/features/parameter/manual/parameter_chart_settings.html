This parameter allows a user to define chart settings.

You can use the user input in a query as follows. Make sure to use the tojson()-function to convert the string to an object:

| paramdefaults
	chart_settings={charttype: "bar", stacked: true }
| source random records=100 type=series
| chart 
	by=[WAREHOUSE, ITEM]
	x=TIME 
	y=COUNT 
	settings = tojson(param("chart_settings", {}))
