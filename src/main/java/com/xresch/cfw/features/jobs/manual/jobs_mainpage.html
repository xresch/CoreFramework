Jobs are providing you with the possibility to schedule tasks to be executed periodically. You can use Jobs to:

  • Alert: Fetch and evaluate data based on thresholds and send alerts based on conditions.
  • Report: Periodically fetch data and report the data to eMail, folder etc...
  • Execute CLI: Execute commands on the server, report the results to a reporting channel.
  • Setup Manual Execution:  Setup Jobs for manual execution by set the to enabled and set a start time far in the future.
  • Trigger From Dashboard:  You can use the Widget 'Trigger Jobs' to allow a user to trigger jobs from a dashboard.
