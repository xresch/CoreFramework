This section documents the standard widgets of the application. Below you find information about the widgets in general. For the documentation of the widgets itself, please refer to the sub pages.

Adding Widgets

To add widgets to a dashboard do the following:

  1. Open Dashboard:  Open the dashboard you have created by clicking the view button.
  2. Click Edit:  Click the "Edit" button that you can find right after the dashboard title. The "Add Widget" button will appear and the "Edit" button will be replaced by a "Done" button.
  3. Add Widget:  Click "Add Widget" and choose a widget from the dropdown to add it to the dashboard. The widget will be added with default contents.
  4. Adjust Widget:  To change your widget, click the small gear icon in the top right corner of the widget while in edit mode.
  5. Saving:  All changes to the widgets are saved to the database implicitly. When you are done and you want to leave the dashboard, click the Done button.

Settings

The widgets that you can add to the dashboard have settings to customize them. When you are in edit mode, all the widgets will have a small gear icon on the top right corner. Click on the gear and choose "Edit" to open the settings for the widget. The settings panel consists in most cases of three parts:

  • Description: A short description of the widget and how to use it.
  • Widget Settings: The settings which are specific to this type of the widget. For some widgets there are no settings and this part will not be available.
  • Widget Default Settings: The settings that are common and shared by all of the widgets.

When you are working with widgets settings:

  • Field Details:  Most fields have a small info icon on the left side. Hover with your mouse on the icon to get some explanation about the field.
  • Two Save Buttons:  There are two save buttons, one of each section of the settings. Please keep in mind that clicking Save will only by applied for the section you are saving.
  • Default Title:  Most of the widgets will have a default title, which is the same as the name of the widget. To remove it make the title field blank.
  • Add Links:  Use the Label Widget to add links to other applications. By changing the widgets color, you can make it look like a button.
  • Adding Custom HTML:  Administrators with the permission "Allow HTML" and/or "Allow Javascript", can add custom HTML/Javascript to the title/footer field to add custom content to the widgets. (This will be overwritten if somebody without the mentioned permissions will edit and save the widget.)
