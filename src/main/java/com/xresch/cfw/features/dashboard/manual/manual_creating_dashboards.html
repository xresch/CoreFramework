The Dashboard List

The dashboard list shows all your own dashboard and dashboards from other users you have access to. Here you can create and manage dashboards. Depending on the rights you have, you have more or less possibilities:

  • Dashboard Viewer:  Can view dashboards that other users have shared. Cannot create dashboards, but might edit when allowed by a dashboard creator. (Tab: Shared Dashboards)
  • Dashboard Creator:  View, Edit and Delete all dashboards of all users, regardless of the share settings of the dashboards. (Tab: Manage Dashboards)
  • Dashboard Admin:  Can view and create dashboards and share them with other users. (Tab: Manage Dashboards)

For each dashboard in the list you can have the following actions(Dashboard Viewers only have the View action):

  • View:  Opens the dashboard.
  • Edit:  Edit the dashboard settings. For editing the dashboard contents, use the Edit button on the dashboard itself.
  • Duplicate:  Creates a copy of the dashboard and saves it to your own dashboards. Will reset the share settings of the dashboard and make it "private".
  • Delete:  Permanently deletes a dashboard.

Dashboard Settings

By clicking on the edit button the Dashboard settings will open. You can provide a name and a description for the dashboard, as well as define how you want to share the dashboard. When sharing a dashboard please consider the following:

  • Share with All:  When you set "Is Shared" to true, it will be shared with all users by default.
  • Share with specific Users:  "Is Shared" has to be true, and at least one user has to be chosen for "Shared with Users" to change the scope from "All Users" to the ones you specify.
  • Editors:  Users specified as "Editors" can always access the dashboard, regardless if "Is Shared" is true or false. These users can change the dashboard content but not these settings itself.
