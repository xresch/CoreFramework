The following image shows an overview of most of the Standard Widgets:

Text

The text widget can be used to add simple plain text descriptions to a dashboard.

Label

The label widget is useful to create headers and vertical labeling for a row of widgets. You can add a link to the label that is opened in a new window when the label is clicked. If you change the labels background color, you can also make a button like widget.

List

With the list widget you can add a ordered or unordered list of items.

Checklist

The checklist widget will add a list with checkboxes. The checkboxes can be selected and deselected even when you are not in edit mode.

CSV Table

This widget allows you to take comma separated values and display them as a table.

Image

Take the image widget and add an image with a URL to your application.

HTML Editor

Create your own HTML Content using this widget.

Website

Show another website on your dashboard. This will only work when the website you want to integrate on the dashboard allows it.

Youtube Video

Add a Youtube video to your dashboard.

Refresh Time

Adds a digital watch to your dashboard to show the last time when the dashboard was updated.

Parameters

Allows to display parameters of the dashboard as input fields for the dashboard user. The user can change the parameters to adjust the data the dashboard displays to his needs.

Replica

Used to copy the contents of a widget from another dashboard. Cannot be used to copy another Replica widget. This can be useful if you want to show the same widget on multiple dashboards and manage it in a central place. Another use case is to enable users to add information to a dashboard(e.g. with a HTML Editor Widget), but should not be able to edit the dashboard displaying the data.
