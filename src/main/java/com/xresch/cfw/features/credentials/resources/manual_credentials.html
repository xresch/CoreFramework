You can store your credentials and access information in a central place under Menu >> Tools >> Credentials. These credentials can then be accessed in the query language using the function credentials(name).

This can also be useful to not have a URL distributed in 100 queries over 25 dashboards, but having it defined in a single place and easy to update in case it ever will change.
