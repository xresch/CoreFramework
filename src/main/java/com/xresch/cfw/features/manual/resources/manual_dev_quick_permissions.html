You can add a permission to the application by creating a new permission object and add it to the database. 

	CFW.DB.Permissions.oneTimeCreate(
new Permission(MyFeature.MY_PERMISSION_NAME, "user")
	.description("The description of the permission.")
	.isDeletable(false),
true,
true);

Afterwards you can check the permissions using the request context:

	if( CFW.Context.Request.hasPermission(MyFeature.MY_PERMISSION_NAME) ) {}
