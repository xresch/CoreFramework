Note:  This is a feature is provided by the base application and is only enabled when needed. It will only appear if the application makes use of it.

The context settings can be found in the menu under "Admin >> Context Settings". This feature provides a possibility to add one or multiple sets of properties for a specific purpose. For example adding multiple system environments that will be available for selection in another part of the application. Please refer to the application specific manual pages to get more details about the context settings available for your application.

Tip: You can as well hover over the info icon on the settings field to get more information about what data you need to enter.
