Time Frame Parameters:

  • earliest: The earliest time in milliseconds from 01. January 1970 00:00:00.000.
  • earliest_Y: The year of the earliest time.
  • earliest_M: The month of the earliest time.
  • earliest_D: The day of the earliest time.
  • earliest_h: The hour of the earliest time.
  • earliest_m: The minute of the earliest time.
  • earliest_s: The second of the earliest time.
  • earliest_ms: The milliseconds of the earliest time.
  • latest: The latest time in milliseconds from 01. January 1970 00:00:00.000.
  • latest_Y: The year of the latest time.
  • latest_M: The month of the latest time.
  • latest_D: The day of the latest time.
  • latest_h: The hour of the latest time.
  • latest_m: The minute of the latest time.
  • latest_s: The second of the latest time.
  • latest_ms: The milliseconds of the latest time.
