The default application contains some functionality to analyze the behaviour and status of the system. This can be found under "Admin >> System Analytics". For a user to be able to access these feature, he will need the permission "System Analytics".

CPU Sampling

The application is taking snapshots of what is going on inside the application. Thanks to that, you can browse the stacktrace of all the methods and see where the application consumes the most time.
The CPU Sampling feature can be found under "Admin >> System Analytics >> CPU Sampling"
