This is a list of questions where you can check your knowledge and find out how to do things in the application. The questions assume you are logged in as a user with admin permissions.

What do you do if you get a request to add a new user to the system?

  1. Open Menu Bar >> Admin >> Manage Users.
  2. Click Create User and fill in the details. If the user is a foreign user(e.g. Active Directory) use a random password and set Foreign to true.
  3. Open the created user and add him to the roles/groups he needs.

What do you do to check what access a user has?

  • Click the stethoscope button to get an audit of the user.
  • Go to the Full Audit Tab and create a report

What do you do to remove a users access to EMP or certain features?

  • Set the user to inactive.
  • Delete the user(might not work if user is created from Active Directory)
  • Remove user from roles/groups to remove certain permissions

What do you do to check if a user has ever logged in?

Go to the User Management and check if the Last Login value is set.

What do you do to check the newest created users?

Go to the User Management and sort by Date Created.

What do you do to export a list of all users?

  • Go to user management, set Display As to CSV, set page Size to All
  • Menu Bar >> Tools >> API, use the API User - fetchData.

What do you do to see all the currently logged in users?

Menu Bar >> Admin >> System Analytics >> Session Overview.

What do you do to get more detailed logs?

Go to Menu Bar >> Admin >> System Analytics >> Log Configuration and adjust the log levels.

What do you do to connect to the database?

  1. Open the log file and copy the database connection link.
  2. Use a Database Tool like DBeaver.
  3. Add a connection using the h2driver and the connection details from the connection link.

What do you do to use a backup of a database?

Replace the database file under {APP_ROOT}/datastore/h2database.mv.db with the file extracted from the backup .zip-File.

What do you do to grant another application access to the API?

  1. Open Menu Bar >> Tools >> API
  2. Click the button "Manage Tokens".
  3. Click on "Add Token"
  4. Optional: Prepend a short name to the token(e.g "acme-app-") to easily identify the token in logs and session overview.
  5. After creating the token edit the token and add the permissions to the token.

What do you do to revoke access of an API token?

  • Edit the token an remove the permission(s) to be revoked.
  • Set the token to inactive
  • Or delete the token

What do you do to check how often an API token is accessing the application?

Go to the log files and search for "apitoken[" .
