Allows the user to select one of the Postgres environments that the user has permission to use. The value of the parameter will be the id of the environment.

Here is an example how to use this to set the environment for the source postgres with a parameter 'database_id': 

| paramdefaults
	database_id=-1
| source postgres 
	environment=param(database_id) 
	query= `
		SELECT *
		FROM ITALIAN_DESSERTS
	`
