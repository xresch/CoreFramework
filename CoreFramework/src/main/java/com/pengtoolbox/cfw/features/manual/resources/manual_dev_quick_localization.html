Register Language Files

Use the following methods to register your localization files. The second parameter will define the context in where the files will be loaded on client side. On server side, always all registered files will be loaded. Therefore make sure you use unique names for your properties. The following is an example on how you can register property files: 

CFW.Localization.registerLocaleFile(Locale.ENGLISH, "/app/example", new FileDefinition(HandlingType.JAR_RESOURCE, "com.example.package", "lang_en_example.properties"));
CFW.Localization.registerLocaleFile(Locale.GERMAN, "/app/example", new FileDefinition(HandlingType.JAR_RESOURCE, "com.example.package", "lang_de_example.properties"));

Localize Strings

You can use the following notation in anything that will be replied in with an AbstractResponse, like HTMLResponse, JSONResponse or PlaintextResponse. The RequestHandler will search the response for the properties and replace them with the localized string. 

	{!example_property_name!}

Server Side - CFW.L()

To retrieve a string and provide a default value for fallback, you can use the method CFW.L():

	CFW.L("example_property_name", "Default Value");

Here an example on how to do it with placeholders:

CFW.L("cfw_dashboard_error_usernotresolved",
	"The the dashboard owner with name '{1}' could not be resolved. Set the owner to the importing user.",
	dashboardObject.has("username"))

Javascript - CFWL()

In Javascript, you can retrieve a string and provide a default value for fallback, you can use the function CFWL(): 

	CFWL("example_property_name", "Default Value");

Language Priority

Multiple languages are used to determine the locale that will be displayed to the user. The application always tries to get a localized value and will fallback to English if the specified languages were not found. Therefore it is recommended to always provide a properties file in English.

  • Prio 1: Browser language of the user.
  • Prio 2: Application language configured on the configuration page.
  • Fallback: When other languages have no match, try to fallback to English.
