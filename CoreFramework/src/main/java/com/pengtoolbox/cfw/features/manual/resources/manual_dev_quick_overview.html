Technologies

The following technologies are used in the CoreFramework(CFW) and provided out of the box:

  • Jetty:  Embedded web server.
  • H2:  The database with the application.
  • javax.mail
  • GSON
  • Bootstrap:  Main CSS Framework, various themes integrated .
  • Font Awesome:  For all these cool icons..
  • JQuery
  • Highlight.js
  • Summernote.js:  Used as the default WYSIWYG editor.

Most important Classes and Interfaces

The following classes and interfaces are the core of the framework:

  • CFW:  Central class through which various functionality can be accessed in a hierarchical manner, for example:
    • Context.Request.getSessionData()
    • Context.DB.Config.getConfigAsString()
    • CFW.Registry.Components.addRegularMenuItem()
  • CFWAppInterface:  Interface to create an application to run with the CFW.
  • CFWAppFeature:  Abstract class that is extended to create features for the application.
  • CFWObject:  Core class to create data objects that are used to:
    • Create database tables
    • Fetch data from the database
    • Save data to the database
    • Generate forms
    • Create REST APIs
  • CFWField:  Core class to define advanced fields for CFWObjects.
  • CFWLog:  The interface for logging CFW standard logs.
  • Response:  The following classes are used to create servlet responses:
    • HTMLResponse
    • JSONResponse
    • PlaintextResponse
