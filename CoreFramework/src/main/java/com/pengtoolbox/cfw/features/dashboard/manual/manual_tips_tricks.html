Resizing and Fitting a Dashboard on the Screen

To fit the dashboard on various screens, you can use the following:

  • Fullscreen Button:  Use the fullscreen button on the top right of the dashboard, to hide the upper part of the dashboard to save space.
  • Browser Fullscreen(F11):  Make your browser display in fullscreen by hitting F11 on your keyboard or using the fullscreen option from the browser menu.
  • Page Zoom:  Change the zoom factor of the page by using Ctrl+Mousewheel or changing the zoom in the browser menu.
  • Combine: Combine the above to create the view you need.

Add extra Space and use Height

When creating dashboards, it is a best practice to size a widget to consume a little extra space. This will prevent that you have to move around a lot of widgets when you have to add more content and the content will fit the widget even when displayed on a small monitor. Also because the dashboard sets the width of the widgets by percent and not absolute, it is a best practice to give a little extra height to a widget to avoid scrollbars when displayed on other monitors.

Refresh Time

Add a refresh time widget for dashboards that are refreshed on a regular basis. This will help you to see of what time the data actually is.

Hack: Multi Duplicate

If you need to duplicate a widget multiple times, you can use this hack that is not really a feature, but is possible because of how the UI reacts to this interaction. Left click on duplicate and drag it slightly away and release the mouse. This will duplicate the widget while keeping the popup open and you can duplicate another time right away.
